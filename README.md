# TAKT-OS/.github

Community health files and the **organization profile README** for [TAKT-OS](https://github.com/TAKT-OS).

| Path | Purpose |
|------|---------|
| [`profile/README.md`](profile/README.md) | Renders on the [organization home page](https://github.com/TAKT-OS) |
| [`assets/takt-os-banner.svg`](assets/takt-os-banner.svg) | Wide banner for the organization profile README |
| [`assets/org-icon.svg`](assets/org-icon.svg) | Source vector icon (not for GitHub avatar upload) |
| [`assets/org-avatar.png`](assets/org-avatar.png) | **512×512 PNG — use this for org avatar upload** |
| [`assets/org-avatar.jpg`](assets/org-avatar.jpg) | 512×512 JPG alternative for org avatar upload |
| [`assets/org-icon.png`](assets/org-icon.png) | Transparent 512×512 PNG (logo only, no background) |

> GitHub accepts **PNG, GIF, or JPG only** — do not upload `.svg` as the organization picture.
> Upload at [Organization profile settings](https://github.com/organizations/TAKT-OS/settings/profile).
